# deepdive canary data

Machine-published nightly by the canary on the deepdive box — one commit per
batch. `report.json` is the last 30 days of ledger rows (trust mix, citation
support, confidence, cost per fixed question), charted live at
https://sprayberrylabs.com/deepdive-canary. Queries, ledger rules, and the
runner live on `master`.
